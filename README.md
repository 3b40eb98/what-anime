# Install 

### homebrew

```bash
brew tap bluuesz/what-anime-re
brew install what-anime-re
```

### npm

```bash
npx what-anime-re
```


# Usage 
### Get Anime By Image File 

`what-anime-re file anime.jpg`

### Get Anime By Image Link 

`what-anime-re link https://anime.com/image.png` <br>
